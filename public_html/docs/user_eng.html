User manual

File upload

File Picker

Press "Choose files" and select appropriate HILLS files. Multiple files can be selected with usage of Ctrl or Shift key.

Drag and drop

Drag and drop chosed files into the empty center of the program area. Multiple files can also be used.

Examples

For educational purposes several example HILLS files are included in the program. Any of them can be picked by clicking on "Examples" button and selecting an example. More information about the simulations are in the accompanied article.

The program accepts HILLS files produced by Plumed 1.3 or Plumed 2. Syntax of files are recognised automatically, nothing has to be manually adjusted. Gaussians written in HILLS files are being sorted according time (in the case of Plumed 1.3 files) or timestamp (Plumed 2.x). For disabling sorting, "sort=false" has to be written into the address line (see further).

Controls buttons

Languages

Button located at right top corner controls a selection of languages. At current version of the program only English and Czech languages are supported.

Play/Stop

Button located at left bottom corner starts or suspends visualization of the data.

Reset

Returns the vizualization on the beginning.

Measure

Opens window with tools for measuring of energy and other quantities (see further).

Speed
0.3 x

Sets speed of visualization. Normal speed (1x) is adjusted to make whole visualization last 10 seconds.

Resolution
128px

Sets size od computational matrix, where potential is deposited. This resolution doesn't affect the resolution of image, but it improve quality of the energy surface.

Export

Opens new window for exporting images. For more information see "Image Export" category

Loop

If it is turned on, the visualization will not stop at the end but loop back to the beginning.

WebGL

Turns off or turns on the usage of WebGL technology. WebGL is much more effective for visualization, but it may not work properly on some devices. Devices without WebGL support have WebGL disabled no matter of this button.

Resize

By holding left mousebutton over this button it is possible to change size of the program screen. If size of the plot area is close to square dimensions, size will be adjusted to exactly square one.

Slider

Control element above the bottom control strip. It displays the current progress throught the visualization and also the progress can be changed by moving the slider to either side.

Axis adjustment
CV rename

Collective variables can be renamed by clicking on them and by writing the new name.

Energy axi units

Energy units on axi title can be changed by clicking on them. Two possibilities are there: kJ/mol and kcal/mol. The program cannot recognize appropriate units from HILLS file, so it has to be set manually.

Image export
Save image

After click on Export button, whole visualization is replaced with an image which can be downloaded through right click and SaveAs.. action.

Image resolution

It is not possible to select dpi value for created image but you can increase the resolution by increasing the size of plot (Resize button).

Exported image is exactly the same as the visualized plot, so any change which you like do (as axi name change or others) must be done before the export.
